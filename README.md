CBBMiner
========

A simple command-line tool to fetch data from Central Bank of Brazil.
It spits out some xml as a result. 

========
Requirements:

  PHP5+

========
Usage:
 
  php init.php -args
  
  -h this help

  --getValues [dd/mm/yyyy] [dd/mm/yyyy] [seriesCode] [seriesCode] ..
